openSSH
Install openSSH server. https://phoenixnap.com/kb/ssh-to-connect-to-remote-server-linux-or-windows#htoc-how-to-install-an-openssh-server
sudo apt-get install openssh-server ii.
manage connection
sudo service ssh restart
sudo service ssh status

Change configuration at:
$ sudo nano /etc/ssh/sshd_config
you need to change lines for password settings:
PermitRootAccess yes
PasswordAuthentication yes
Check what port you are using (fx 8822)
Port 8822

Allow that port through firewall
$ sudo ufw allow 8822
 
Reset passwd if necessary with command
$ passwd
